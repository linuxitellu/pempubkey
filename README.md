# pempubkey

Easy create `pubkey.pem`

```bash
git clone https://github.com/linuxitellu/pempubkey.git
cd pempubkey
./pubkeygen
```

Will automatically create `pubkey.pem` File

```bash
cat pubkey.pem
```
